## Tier 4 — Pinterest-specific (long-tail passive)

Pinterest allows raw affiliate links in pins as of 2024. Every aesthetic still pinned with an Amazon link is a passive earner forever.

Process:
1. Generate / select a 4:5 vertical still
2. Upload as a standard pin
3. Set destination URL = Amazon affiliate link for the dominant prop (candle, book, jewellery)
4. Title: literal description ("Brass Three-Arm Candleholder in a Gothic Library")
5. Description: 2 sentences + 5 keyword-rich hashtags
6. Pin to relevant board: "Dark Academia Interior", "Gothic Reading Nook", "Autumn Aesthetic"
7. Aim for 5–10 affiliate pins/day in addition to non-affiliate content pins (algo punishes 100% affiliate boards)

Sample products to seed Pinterest with day 1:
- Brass three-arm candleholders (Amazon search → top 5 picks)
- Leather-bound classic literature bundles
- Smith-Waite tarot deck
- EB Garamond patterned journals
- Antique-looking fountain pens
- Velvet throw blankets (gothic colourways)
- Brass bookmark sets
- Gothic-style wax seals
