# Pinterest Seed List — 30 affiliate-ready pins

> Each pin = one Amazon UK affiliate link (or Awin merchant link once approved). Aim for 5–10 affiliate pins/day mixed with 5–10 non-affiliate aesthetic pins (the algo penalises 100%-affiliate boards).
>
> **Workflow:** open Pinterest → Create Pin → upload image (generate in Colab from the matching `style.md` prompt subset) → paste title → paste description → set destination URL (raw `amzn.to/...` Associate-shortened link — Pinterest allows it since 2024) → assign to board.
>
> **Replace `tag=XXXX-21` in URLs with your Amazon Associates UK tracking ID once approved.** Pinterest will respect the tag and credit you.

## Boards to create first (8)

Create these boards before pinning anything. Each board needs a 4:5 cover pin matching the aesthetic.

1. **The Reading Nook** — interior, candles, books, desks
2. **Gothic Interior** — broader home decor, dark academia spaces
3. **Dark Academia Outfits** — fashion-focused, GRWM stills
4. **Tarot & The Occult** — cards, altars, witchy goods
5. **Stationery & Letters** — fountain pens, journals, wax seals
6. **Autumn Aesthetic** — seasonal, dried flowers, mood

## Posting cadence

- **Days 1–7:** seed all 30 pins, ~5/day across boards
- **Days 8+:** 3–5 new pins/day mixing affiliate + non-affiliate aesthetic stills (the algo rewards "fresh pins" — even a re-upload counts)
- **Idea pins** (Pinterest's video format) on best-performing affiliate products from week 2 onward — usually 3× the engagement of static pins

## SEO inside the title + description

Pinterest is a search engine. Each title should literally describe what's in the image AND include 2–3 keywords someone would search. The "tap to find" / "what I read by" phrases are CTAs, not SEO. The SEO carries in the keyword-rich descriptors ("brass three-arm candleholder", "Smith-Waite tarot deck", "velvet altar cloth").
