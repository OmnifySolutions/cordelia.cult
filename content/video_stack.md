# Video Stack — Operator Guide

> Image-to-video tooling for Cordelia. Read this before each Sunday batch. Updated 2026-05-18.

## Decision tree — which tool for this clip?

```
Start: I have a Cordelia still and need motion.
│
├─ Does the clip need lip-sync to a voiceover?
│  └─ YES → SadTalker (colab/video_realistic.ipynb Section B)
│
├─ Does the clip need a very specific facial micro-expression I can't prompt for?
│  └─ YES → LivePortrait (colab/video_realistic.ipynb Section A)
│
├─ Do I still have Seedance daily credits?
│  └─ YES → Seedance 2.0  ← DEFAULT, USE THIS
│
├─ Do I need dramatic cinematic camera motion Seedance can't deliver?
│  └─ YES → Kling AI (accept the watermark, crop in CapCut)
│
├─ Burned both daily quotas, still need clips?
│  └─ YES → Wan 2.2 on Colab T4 (slow but unlimited)
│
└─ Nothing else works → re-generate the still with better composition and retry
```

## Daily quota math

| Tool | Daily free credits | Cost per 5-sec clip | Max clips/day | Resolution | Watermark | Commercial OK |
|---|---|---|---|---|---|---|
| Seedance 2.0 | 100 | ~20 | ~5 | 1080p | No | Yes |
| Kling AI | 66 | 25 | ~2-6 | 720p | Yes | Limited |
| Wan 2.2 (Colab T4) | Unlimited | ~5-10 min compute | ~5-10 | 720p | No | Yes (Apache) |
| LivePortrait (Colab T4) | Unlimited | ~2 min compute | ~20 | 720p | No | Yes |

**Realistic Sunday batch capacity:** ~5 Seedance + ~2 Kling + 1 LivePortrait or SadTalker = 8 source clips. Enough for 7 daily TikTok pieces plus 1 spare.

## Tool-by-tool

### 1. Seedance 2.0 — primary

- **Signup:** https://seedance.ai (or https://seedance.tv) — Google sign-in, no credit card.
- **Upload flow:** Image-to-video tab → upload Cordelia still → motion prompt → generate.
- **Prompt format that works:**
  ```
  [camera move], [subject motion], [ambient detail], cinematic, 4k
  ```
  Examples:
  - `slow dolly in, hair drifts gently in candlelight, dust motes float, cinematic, 4k`
  - `subtle parallax pan left, candle flame flickers, page corner curls, cinematic, 4k`
  - `static shot, blinks slowly, exhales fog, cinematic, 4k`
- **Avoid:** complex multi-action prompts ("she turns, picks up book, smiles") — Seedance handles ONE motion well, not three.
- **Output:** MP4, 1080p, ~5s, no watermark. Drop straight into `content/[week]/source/`.

### 2. Kling AI — secondary, cinematic motion

- **Signup:** https://klingai.com — Google sign-in, no credit card.
- **When to use over Seedance:** anything needing dramatic camera moves (orbit, crane, push-pull, anamorphic widescreen feel). Kling's motion is genuinely best-in-class even on free.
- **Quota strategy:** 66 credits/day, 5-sec clip = 25 credits → 2 standard clips/day. Don't try 10-sec clips (50 credits each, halves daily output for marginal gain).
- **Prompt format:** same as Seedance but lean harder into cinematic vocabulary — `anamorphic lens flare`, `golden hour rim light`, `35mm film grain`, `dolly zoom`.
- **Watermark handling:** Kling places a small logo bottom-right. In CapCut, import clip → scale to 105-108% and reposition slightly up-left to crop the corner.

### 3. Wan 2.2 on Colab — unlimited backup

- **Notebook:** https://github.com/theelderemo/wan2.2-google-colab (pre-configured for free T4)
- **When to use:** Seedance + Kling daily quotas both spent and you still need clips. Or testing prompts at zero cost before burning Seedance credits on the final.
- **Constraint:** free T4 can only run Wan 2.2 5B model (TI2V-5B), not the 14B. Quality is ~Kling-tier, not Seedance-tier. ~5-10 min per clip vs Seedance's ~30 sec.
- **Workflow:** open notebook → connect T4 → upload still to `/content/` → set prompt → run all → download MP4.

### 4. LivePortrait — precision face control

- **File:** `colab/video_realistic.ipynb` Section A
- **When:** the still has a specific facial expression you want preserved exactly, with only natural blinks/micro-movements added. Seedance and Kling will sometimes morph the face — LivePortrait won't.
- **Inputs:** one Cordelia still + a "driving video" of any face moving (provided in the notebook). Output is your still animated with the driving video's motion.
- **Limitation:** no audio, no big movements, no scene change. Just face animation.

### 5. SadTalker — lip-sync for voice posts

- **File:** `colab/video_realistic.ipynb` Section B
- **When:** the 1-in-5 tentpole voice post where Cordelia "speaks" the caption.
- **Inputs:** one Cordelia still + a WAV file (generated by Edge TTS from the post script).
- **Limitation:** mouth movement is decent but not perfect; works best with face filling the frame and short, well-paced audio (<15 sec).

## Common failure modes

1. **Face morphs between still and video.** Tool added too much motion. Fix: simpler prompt ("static shot, slow blink, hair drifts") or switch to LivePortrait.
2. **Video looks stiff/static.** Still is too tightly cropped — image-to-video needs negative space to animate. Re-generate still with more headroom.
3. **Burned daily quota on bad outputs.** Always test prompts on Wan 2.2 (free, unlimited) first when uncertain, then commit Seedance credits to the winner.
4. **Kling watermark visible after CapCut crop.** Increase zoom to 110% and shift up-left more. If the composition won't tolerate that crop, regenerate on Seedance instead.

## Phase upgrade path

- **Phase 0–1 (days 1–45):** free tier only. Quota = 8 clips/day = enough for 3 TT pieces + 3 IG + cross-posts.
- **Phase 2 (day 45+, if revenue ≥ £100):** consider Kling paid (~£8/mo) for unwatermarked + more credits. Cheapest leverage upgrade.
- **Phase 3 (day 90+, if revenue ≥ £400):** consider Seedance paid tier if free quota is the only thing blocking 10+ clips/day.
- **Phase 4 (day 180+):** used RTX 3090 unlocks local Wan 2.2 14B (Seedance-tier quality, infinite throughput, zero ongoing cost).
