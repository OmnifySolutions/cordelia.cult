## Step 0 — Prep (5 min, before touching any platform)

### Dedicated email
Create a brand-new Gmail address — **don't use your personal email**. Suggestions:

- `cordelia.vale.cult@gmail.com`
- `cordelia.cult.studio@gmail.com`
- `vale.cordelia@gmail.com`

Use this email for **all 5 platforms + Linktree + Beehiiv (later) + every affiliate program**. Keeps your real identity separate from the brand and makes the brand sellable later (handing over an email + handle = handing over the business).

### Password manager
If you don't already have one: install **Bitwarden** (free, cross-platform). Generate a unique 20-char password for each platform — never reuse.

### Phone number
TikTok and Instagram both require SMS verification. Use your real phone — these don't get spam, and it's needed for monetisation features later. One phone can verify multiple accounts.

### 2FA
Enable 2FA on the Gmail account first (Authenticator app, not SMS). Then enable on every platform that supports it. **TikTok bans without warning if security looks weak** — 2FA reduces ban risk.
